# Algoritmos Genéticos: Trabalho final

## Equipe: Diogo P. de L. Carvalho, José D. A. Sales e Mayllon E. P. S. Silva

### Introdução
<p>Este repositório contém o projeto final do curso de Algoritmos Genéticos, que visa otimizar a formulação de vidros à prova de bala. Utilizando dados de uma base de dados contendo diversos compostos de formação de vidros, nosso objetivo é melhorar significativamente as propriedades de resistência balística do vidro enquanto mantemos os menores custos possíveis de produção. Este projeto combina a ciência dos materiais com técnicas avançadas de otimização baseada em algoritmos genéticos para encontrar soluções inovadoras e economicamente viáveis. Os dados dos compostos a serem trabalhados estão disponíveis no “Glasspy” [1], esse módulo por sua vez, não oferece as informações necessárias para trabalharmos com os preços dos compostos, para solucionar esse problema, utilizamos os preços do “Sigma Aldrich”[2], uma empresa referência no fornecimento de produtos para pesquisa científica</p>
